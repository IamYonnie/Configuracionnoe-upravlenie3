**Общее описание**
Проект реализует ассемблер для учебной виртуальной машины. Ассемблер преобразует текстовые команды в формате CSV в бинарный машинный код, который может выполняться виртуальной машиной.

**Основные компоненты**
**assembler.py** - основной файл ассемблера, который:
- Читает CSV-файл с программой
- Преобразует текстовые команды во внутреннее представление
- Кодирует команды в бинарный формат согласно спецификации УВМ
- Сохраняет результат в бинарный файл
**uvm_defs.py** - содержит спецификацию УВМ:
- Описание всех команд (load - загрузка константы, read - чтение из памяти, store - запись в память, cmp_ge - сравнение >=)
- Коды операций и битовые раскладки для каждой команды
- Тестовые примеры из спецификации

Как использовать:
Скачать все файлы и запустить программу "run_tests"ю
